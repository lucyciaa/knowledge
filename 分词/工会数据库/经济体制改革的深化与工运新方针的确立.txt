1993年10月24日至30日，中国工会第十二次全国代表大会在北京隆重召开。大会的主要任务是全面贯彻党的十四大精神，以邓小平建设有中国特色社会主义理论和党的基本路线为指导，系统全面地总结自中共十一届三中全会以来的十五年工人运动和工会工作的伟大实践和基本经验，决定今后一个时期工会工作的方针和主要任务。由地方工会、产业工会、中共中央直属机关、中央国家机关工会分别选出的1600名代表和296名特邀代表及21名港澳地区工会界代表出席大会，代表着全国2亿职工和1. 02亿工会会员。党和国家领导人江泽民、李鹏、乔石、李瑞环、朱镕基、刘华清、胡锦涛等出席了开幕式。大会由全国总工会主席倪志福主持。江泽民到会作重要讲话。中共中央政治局常委、中央书记处书记胡锦涛代表党中央、国务院发表了《我国工人阶级的伟大使命》的祝词。朱镕基作经济形势报告。全国总工会副主席、书记处第一书记张丁华代表全国总工会第十一届执委会作了题为《团结动员全国职工发挥主力军作用，为夺取有中国特色社会主义事业的更大胜利而奋斗》的工作报告。尉健行致闭幕词。
大会通过了关于工作报告的决议、关于《中国工会章程 (修正案)》的决议、关于财务工作报告和经审工作报告的决议，选举出全总十二届执委241人、主席团委员34人，推选出书记处书记10 人。尉健行当选为全总主席，张丁华、杨兴富、李奇生、刘珩、江家福、张国祥、方嘉德、薛昭鋆 (女)、滕一龙为副主席。张丁华被推选为十二届执委会书记处第一书记。大会表彰了基层工会模范职工之家、工会小组、模范职工小家和优秀工会工作者、荣誉工会积极分子。
中共中央总书记江泽民在会前接见大会主席团常务主席时发表重要讲话。江泽民指出，建设有中国特色社会主义是人类历史上最辉煌的事业，也是中国工人阶级和广大人民群众自己的事业。在社会主义现代化建设的关键时期，我们要在20世纪末初步建立起社会主义市场经济新体制，实现中共十四大确定的宏伟目标和任务，就必须全心全意依靠工人阶级，发挥工人阶级的主力军作用。工人阶级要认清自己的历史使命，紧密团结在党的周围，艰苦创业，奋发努力，为改革和建设不断作出新贡献。他充分肯定了改革开放十五年来，各级工会组织的工作成绩，并对其提出了更高要求。江泽民指出，当前，深化企业改革、转换国有企业特别是大中型企业的经营机制，是摆在我们面前的一个重大课题。各级工会要团结和动员广大职工积极投身改革，推动改革不断深入。要及时反映职工群众的意见和要求，切实维护职工的合法权益。要积极探索有中国特色社会主义工会工作新路子，努力开创工会工作的新局面。各级党委要加强和改善对工会工作的领导，支持工会依照宪法、法律和自己的章程独立自主地开展工作，使工会真正成为党和政府密切联系广大职工群众的桥梁和纽带。
胡锦涛在祝词中高度评价了中国工人阶级的历史地位和作用，指出了中国工人阶级在新时期的伟大使命，强调了在社会主义市场经济条件下，工会的地位和作用更加重要，工会的任务和责任更加重大，并对工会的改革和建设提出了具体要求。这对工会运动的新发展，具有重要指导意义。
张丁华在工会十二大的工作报告中，回顾了中共十一届三中全会后十五年来工人运动的伟大成就，总结了工会工作的基本经验。报告指出，中国工会十五年来，在理论上有新的突破，在实践上有新的创造，工作领域有新的扩展，社会作用有新的增强。但工会工作中也存在差距和不足，即在发展社会主义市场经济的形势下，思想观念不完全适应客观形势的要求，对经济关系、劳动关系研究不够，多层次、多渠道参与不够; 工会组织的群众化、民主化建设还有差距，需要进一步改进工作作风，密切与职工群众的关系。报告把建设有中国特色社会主义工会工作的基本经验概括为九条: ( 1) 必须自觉地接受中国共产党的领导，并在党的领导下，独立自主地开展工作，两者统一于工会工作的全过程; ( 2) 必须坚定不移地贯彻党的基本路线，关键是以经济建设为中心不动摇; ( 3) 必须坚定不移地推动党的全心全意依靠工人阶级根本方针的贯彻落实; ( 4) 必须坚持解放思想、实事求是的思想路线，不断推动工会理论和实践的发展; ( 5) 必须坚持“两个维护”的统一，全面履行工会四项社会职能; ( 6) 必须坚持与政府的相互合作与支持，加强对政府工作的民主参与和社会监督，成为国家政权的重要社会支柱; ( 7) 必须密切联系群众，成为中国共产党联系职工的桥梁和纽带，成为职工群众利益的重要代表者和维护者; ( 8) 必须以增强基层工会活力为中心环节，加强工会自身的改革和建设，努力实现群众化、民主化，把工会建设成为职工群众信赖的职工之家; ( 9) 必须坚持“独立自主、广泛联系”的方针，高举“和平、发展和工人权益”的旗帜，发展同各国工会和国际工会组织的交往与合作关系，善于吸取和借鉴外国工会有益的经验，增强有实质内容的交流与合作。

(二) 《中华人民共和国劳动法》的颁布实施
在党的十四大精神指引下，经济体制改革步伐加快，非公经济如雨后春笋般发展起来，国有企业也加紧了适应市场经济要求的改革。这时的经济关系和劳动关系都在急剧深刻地发生变化，劳动关系变得更加复杂。但是，“由于我们国家劳动法律、法规还不完备，在一些地方和企业，特别是在一些外商投资企业中，存在严重侵犯职工合法权益的问题，有的不与职工签订劳动合同，随意辞退职工; 有的压低、克扣职工工资、延长工时，强迫职工加班加点; 有的拒绝为职工提供必要的劳动保护，接连发生重大伤亡事故; 还有的对职工滥施惩罚、侮辱人格，侵犯职工人身权利，造成极为严重的后果，恶化了企业劳动关系，影响了企业发展和社会稳定”。 这一客观形势要求必须及时地强化法律调控手段，市场经济呼唤《劳动法》尽早出台。
党的十一届三中全会召开前夕，邓小平指示应尽快制定《劳动法》。邓小平在1978年12月中央工作会议的闭幕会上指出， “为了保障人民民主，必须加强法制”“应该集中力量制定刑法、民法、诉讼法和其他各种必要的法律，例如工厂法、人民公社法、森林法、草原法、环境保护法、劳动法、外国人投资法等等”“做到有法可依，有法必依，执法必严，违法必究”。八届全国人大常委会任期内， 《劳动法》的制定被列为建立社会主义市场经济法律体系框架中重要的一环。
1994年7月5日，全国人民代表大会常务委员会第八次会议审议通过《中华人民共和国劳动法》。国家主席江泽民签署第28号主席令颁布了这部法律，自1995年1月1日起施行。12日，全总召开电话会议，要求各级工会组织认真学习、贯彻《劳动法》。13日，全总和劳动部在京联合举行《劳动法》颁布实施座谈会。
《劳动法》是社会主义市场经济的产物。其根本立法宗旨是调整劳动关系，维护劳动者权益。它从我国宪法的基本原则出发，对劳动者的权利、义务和权益保护等作出了明确而具体的规定，对用人单位和劳动者提出了一系列行为规范，将劳动关系的调整纳入了法制轨道。
根据《劳动法》规定，劳动者享有平等就业和选择职业的权利，取得劳动报酬的权利，休息休假的权利，获得劳动安全卫生保护的权利，接受职业技能培训的权利，享受社会保险和福利的权利，提请劳动争议处理的权利以及法律、法规规定的其他劳动权利。《劳动法》不仅明文规定了劳动者享有的各项权利，而且建立、健全了保护劳动者合法权益的法律机制，任何侵犯劳动者权利的人都要负相应的法律责任，并且劳动者也可以通过向有关部门反映、提起仲裁和诉讼等程序来捍卫自己的合法权利。
《劳动法》的颁布，使我国的劳动立法迈上了一个新的台阶。使有关机关处理劳动争议有法可依，劳动者与用人单位也能够明确双方的权利义务，对劳动者和用人单位均有裨益。《劳动法》颁布后，国务院、劳动部均先后颁布了很多劳动法规和规章，以配合劳动法的实施。
《劳动法》第七条规定: “劳动者有权依法参加和组织工会。工会代表和维护劳动者的合法权益，依法独立自主地开展活动。”明确了工会是劳动者合法权益的代表者和维护者，维护劳动者的合法权益是工会的法定职责。明确工会的法律地位和职责，是发展社会主义市场经济过程中国家、企业、职工三方利益格局日益清晰化的客观趋势，是劳动关系发展变化的必然结果，也是广大职工群众的迫切要求和愿望。可以说，《劳动法》的颁布为工会工作总体思路的确立提供了基础。
(三) 全总第十二届二次执委会议与工会工作总体思路的确立
《劳动法》颁布后不久，1994年12月8日，全国总工会第十二届二次执委会议召开。会议期间，中共中央政治局常委胡锦涛代表党中央作了题为《全面加强工会建设，在深化改革、扩大开放、促进发展、保持稳定中发挥作用》的重要讲话，明确提出: 要善于在改革开放的新形势下认识自己、加强自己、提高自己，全面加强工会的思想建设、组织建设、作风建设，努力把中国工会建设成为中国共产党领导的、用建设有中国特色社会主义理论武装的、能够带领职工群众为实现党的任务英勇奋斗的、思想上政治上组织上坚强的、深受广大职工信赖的工人阶级群众组织。这为搞好工会改革和建设，做好今后的工作，明确了目标，指明了方向。
12月12日，中华全国总工会主席尉健行作了题为《以贯彻实施〈劳动法〉为契机，推动工会改革和建设，在改革发展稳定中更好地发挥作用》的讲话，提出了新时期工会工作的总体思路，即“以贯彻实施《劳动法》为契机和突破口，带动工会各项工作，推动自身改革和建设，努力把工会工作提高到一个新水平，在改革、发展、稳定中更好地发挥作用”。讲话强调各级工会干部要有很强的责任感，必须始终坚持把贯彻实施《劳动法》放到全党全国工作的大局中来看待和处理，自觉地为实现大局服务。总的要求就是态度要积极、步骤要稳妥、工作要扎实。这次会议还通过了《全国总工会关于贯彻实施〈劳动法〉的决定》。
工会工作总体思路是对社会主义时期工会理论和实践的重大突破，其实质要求是: ( 1) 以贯彻实施《劳动法》为契机，进一步突出工会的维护职能，明确维护是工会的基本职责，明确工会代表和维护职工群众利益的社会地位; ( 2) 工会对职工利益的维护，应是两个维护的统一，即维护全国人民总体利益和维护职工具体利益的统一 (这是维护的指导原则)，维护职工政治权利和经济利益的统一 (这是维护的具体内容) ; ( 3) 工会为经济建设中心服务的着力点是做好人的工作，是保护和调动职工的积极性来推动经济发展; ( 4) 突出维护职能，带动工会其他职能的履行，促进工会各项工作上新水平; ( 5) 充分运用《劳动法》赋予工会维护职工利益的权利和义务，抓住平等协商和签订集体合同这个关键环节 (或称“牛鼻子”) ，发挥工会在整体上维护职工利益的作用。
继1992年《工会法》明确提出“工会在维护全国人民总体利益的同时，维护职工的合法权益”的职责之后，工会工作总体思路的提出更进一步强化了工会维护职工权益的职能，并提高到基本职责的高度，强调要通过维护职工合法权益这个侧重点，来保证党的基本路线的贯彻实施，为国民经济持续、快速、健康发展创造良好的社会环境。
工会工作总体思路的提出，从理论上回答了长期以来有所争论的理论难题，即社会主义时期工会存在的客观必然性问题，这就是: 在社会主义初级阶段，工会组织赖以存在的客观基础是劳动关系。明确了工会的现实身份和地位，即相对于劳动关系层面，工会在调整其主体双方的利益矛盾时，是作为劳动者劳动权益的代表身份出现的。这为突出工会工作重点提供了理论根据。
工会工作总体思路的提出，明确了工会工作在党和国家工作大局中的位置，即在全党全国工作的大局中，工会工作是个局部工作，贯彻实施《劳动法》也是个局部工作。局部工作要服从和服务于全局。贯彻实施《劳动法》，处理各种劳动关系和利益矛盾，都不能离开大局，一个重要的前提就是要有利于改革、发展和稳定，这是工会工作的归宿点。工会服务于党的中心任务的主要手段是在维护全国人民总体利益的同时，更好地表达和维护职工群众的具体利益。
工会工作总体思路对工会自身改革和建设提出了新的要求。工会必须加强自身建设，从精神状态、思想观念到运行机制、工作方法乃至组织体制和工作格局，都将有一个较大的转变。工会改革不仅要从认识上、理论上提出问题去加以探讨和研究，有一个通盘的规划和安排，而且必须从实际出发，根据实际需要去不断推进。什么方面不适应、不合理，就改什么方面。如果在制度上、体制上、机制上、业务上跟履行维护职能的要求不吻合，那就是最现实的自身改革的内容; 如果在方法上、作风上、人员素质上不适应履行维护职能的要求，那就是最现实的加强建设的内容。
