从1840年鸦片战争开始，西方列强的入侵，彻底打破了中国传统社会原有的文明进程，也对中国近代工业的发展产生了一定的刺激作用。首先，西方列强让清政府认识到了“船坚炮利”的意义和价值，而这背后最直接的支持就是西方国家的近代工业体系，促使清政府开始推动中国近代工业的发展; 其次，西方列强为了能够在中国谋求殖民利益，其中就包括设立工业厂矿，这在一定程度上也刺激了中国近代工业的发展; 最后，就是民族资产阶级的崛起，在资本丰厚利润的刺激下和实业救国理念的支持下，民族资本家开始投资设厂推动了民族工业的发展。以上三种因素的交互作用，推动了中国近代工业的产生和发展。
从近代工业的发展进程来看，一般认为最先出现的是外国资本工业，然后是清政府官僚资本工业，最后是民族资本工业。从近代工业的主导发展力量或推动力量来看，中国的近代工业发展，从同治初年开始算起，直到民国初，基本上可以分为五个时期:
第一个时期是“军用工业时期”，从同治元年 ( 1862 年) 至光绪七年 ( 1881年) ，约20年。中国新工业滥觞于军工，如同治四年 ( 1865年) 曾国藩设江南造船厂，同治五年 ( 1866年) 左宗棠设福州船政局，同治六年 ( 1867年) 李鸿章设江南制造局，光绪三年 ( 1877年) 丁宝桢设四川兵工厂。当时朝野上下很少注意制造商品之工业，唯有李鸿章奏设江南制造局折中，有“数十年后，农贾必有仿造洋机以求利益者”之语，薛福成治平六策奏疏中有奖励制器造船两条，因此其眼光较为长远。但这个时期，主要是购买西方的机器，借助外国工程师，然后训练若干中国工头工人，并且将总办大权委派给一些官僚。
第二个时期是“官督商办时期”，从光绪八年 ( 1882年) 至二十年 ( 1894年) ，约13年。1882年，李鸿章奏请在上海试办机器织布局，开官办工业先河，但到光绪十六年 ( 1890年) 才实行筹备，光绪十九年 ( 1893年) 该厂毁于大火，最终未能成立。在这段时期，工业最初大都是官办，但因业绩不佳，逐渐改成官督商办，这大致是在光绪二十年 ( 1894年) 之后了。
第三个时期是“外人兴业时期”，从光绪二十一年 ( 1895 年) 至二十八年( 1902年) ，约8年。这一时期也是华商觉悟时期，《马关条约》准许了外国商人在中国通商口岸设立工厂，而政府却对于振兴工业没有一定的计划，这个时期最值得纪念的，是南通大生纱厂和上海商务印书馆的成立，而在该时期的中后期开始，政府才出台奖励工业的政策。
第四个时期是“政府奖励及利权收回时期”，从光绪二十九年 ( 1903 年) 至宣统三年 ( 1911年) ，约9年。光绪二十九年七月政府设立商部 (三十二年改为农工商部)，以载振为尚书，陈璧、伍廷芳为左右侍郎，奏定商律及公司注册章程，一时官吏提倡于上，绅商响应于下，收回权利之声通行国内; 光绪三十一年 ( 1905年) 因美国禁止华工，开始出现了抵制美货、提倡国货，此时较为重要的，有袁世凯在天津设立的工艺总局，商部在京师设立的劝工陈列所及奏设的各省高等实业学堂; 光绪三十二年 ( 1906年) 商部奏定奖给商勋章程，鼓励制造新器，学部考验游学生设工商科进士学位; 光绪三十三年 ( 1907年) 农工商部奏定华商办理实业爵赏章程，对办1000万元以上实业者赏男爵，2000万元以上者赏子爵; 宣统二年 ( 1910年) 端方在南京举行南洋劝业会为60年仅见之盛举。这种以爵位奖励创办实业者的做法，一扫数千年来的贱商陋习。
第五个时期就是“自动发展时期”，民国元年 ( 1912年) 至十年 ( 1921年) 。就政府对待实业的态度及影响来说，在这60年中，通商口岸的工业借外国势力的庇荫，得到一定程度的自由发展，分沾了欧洲战争的一些利润，而欧洲战争结束后的一段时期，算是中国工业恐慌时期，铁厂积货如山，无人过问，甚至闭炉停机，纱厂结账大多无利。上海数十年的三大油厂竟同年倒闭。
鸦片战争后中国近代工业的发展仍然是非常缓慢的，其原因首先是社会各界在工业进程中的不合作，当时的时评者就非常敏锐地指出，“军用工业时期国人犹在梦中，可置不论，若官督商办以至政府奖励之三时期，则官倡于上而社会退缩不前，故其效甚微。自动发展时期，社会似有觉悟矣，而官吏军人之又从而挠之抑之，故其效仍不大。清末之官贤而民不肖，民国之民贤而官不肖，中国之政府与人民对于工业遂竟无一日之通力合作，五十年之光阴匆匆过矣”。当然，还有一个重要的外部因素，就是帝国主义对中国工业发展的压迫，他们侵略中国，本来就不是想让中国走向现代化，而只是企图让中国成为他们的殖民地。
中国近代铁路工业的发展大致可以反映出中国近代工业发展的历程和状况。在中国大地上最早出现的铁路是1876年英美等国商人在上海擅自建筑的吴淞铁路，长13公里，经营一年。清政府慑于沿路人民的反抗斗争，同时也为避免类似情况再度发生，在1877年高价收回，即予拆毁。清政府在1880年至1894年间，共计修建铁路447公里;1895年至1900年间，扣掉义和团反帝斗争中扒毁的，新设铁路约700公里。帝国主义在19世纪掠取的铁路权益，进入20世纪才加紧转化为现实，俄国全力加速修筑中东铁路，1903年7月该路正式营业; 德国的胶济铁路于1904年建成; 法国在1903年至1909年建筑滇越铁路; 英国强行建筑了道清铁路; 日本在日俄战争期间擅自在东北地区建筑了安奉、新奉两线，又在1909年至1912年间修建了吉长铁路。
总计在20世纪头11年里，中国增建了约8200公里铁路，其中帝国主义直接建筑的有3700余公里，占46% ; 通过向清政府提供贷款而间接建筑的有3300公里，占40% ; 中国自己建筑的，包括京张铁路在内，约1200公里，只占14%而已。1912年至1927年间，北洋军阀统治时期增修了3800公里铁路，其中有三分之二在东北，这些铁路又有一半以上是在日本控制下修建或强行修建的。
